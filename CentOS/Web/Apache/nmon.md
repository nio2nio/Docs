### Install nmon
```shell
$ sudo yum install -y nmon

$ nmon
# t = Top Process Stats
# c = CPU by Processor
# l = Longer term CPU Averages
# m = Memory and Swap Stats
# j = JFS Usage Stats
# h = Help Information
# n = Network Stats
# d = Disk I/O Stats
# o = Disk Busy Map
# k = Kernel Stats and Load average
# v = Virtual Memory Stats
# g = User defined disk groups
# v = Verbose Simple Checks
# b = Black and White Mode
# N = NFS
```
