### CLI Usage
```shell
# Login
$ heroku login

# Using an Http Proxy
# For Unix
$ EXPORT HTTP_PROXY=http://proxy.server:port
or
$ EXPORT HTTPS_PROXY=http://proxy.server:port
$ heroku login

# For Windows
$ set HTTP_PROXY=http://proxy.server:port
or
$ set HTTPS_PROXY=http://proxy.server:port
$ heroku login
```
